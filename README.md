# 감정일기
### 프로젝트 개요
현재 COVID 19로 인한 거리두기로 사람들은 친구, 친척들을 자주 만나지 못하고 혼자있는 시간이 많아졌다. 그래서 자신의 감정을 다른사람과 나누지 못하고, 혼자 생각하는 시간이 많아지다보니 Corona-Blue와 같은 우울증을 겪기도 한다. 이를 해결하기 위해 자신의 하루를 다른사람들과 공유하고, 감정을 치유 받을 수 있는 서비스를 만들고자 했다. 또한 만약 유저가 자신이 일기를 작성하고도 자신이 어떤 감정을 가지고 하루를 보냈는지 확실히 모를때 인공지능 모델이 유저의 감정을 알려주는 서비스를 기획했다.

### 서비스 구조
- 모든 사람의 일기를 볼 수 있는 화면(인스타와 비슷하다.)
- 유저가 가장 최근에 적은 일기의 감정과 비슷한 감정을 가진 일기들만 볼수있는 화면
- 유저가 가장 최근에 적은 일기의 감정과 반대의 감정을 가진 일기들만 볼수있는 화면
- 자신의 일기들만 모아 볼 수 있는 화면
- 자신이 언제 일기를 적었었는지 알 수 있는 캘린더 화면
- 일기를 작성할수 있는 화면 -> 여기서 인공지능이 자신의 감정을 알려주면 좋겠다면 체크를 하면 인공지능이 알려준다. -> 모델 만들기 실패. 추후 보완 예정
